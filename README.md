# Attendance Management System Blog Website

A deployable React + Vite + Tailwind CSS website to showcase the **Attendance Management System** project.

## Run locally

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm install
npm run build
```

The production files will be generated inside the `dist` folder.

## Deploy options

You can deploy this easily on:
- Vercel
- Netlify
- GitHub Pages

## Tech stack
- React
- Vite
- Tailwind CSS
